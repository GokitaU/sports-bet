About myself and about this web service project

    Betting service which I'm presenting you is a result of effort made to submit final project of CodersLab bootcamp. I learned technological aspect which were implemented in this work. The idea was to design it in REST API concept, but as a novice I realize it may not be perfect. List of the covered subjects is at the same time list of my competencies as a Java developer

      Key technology and tools I used in my work included:
    • Spring Boot framework (in particular Security for authentication, Data, Batch to feed database from csv, Hibernate to represent ORM of my model, and MVC)
    • MySQL database design (managed mainly through MyWorkbench)
    • In some service there is unit testing, but I still work to expand a bit this scope
    • Front-end is combination of Bootstrap templates modified to my needs and vision
    • There are also potentially good places to introduce multi-threading, which I'm going to do as a part of further improvements and self-development
